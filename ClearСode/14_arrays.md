Язык Python

1) В HashTable определил определил __iter__ и __next__, что позволило перебирать элементы через использование "for ... in ..." 

2) Для получения элементов в массивах использую "for ... in ..." 
например:
    for user in users:
        ...

3) В случае если если надо получить и индекс элемента, это можно сделать "безопасно" с помощью "enumerate"
    for idx, user in enumerate(users):
        ...

4) Массив в Python можно использовать как стек.
Например, в задаче на подсчет закрытия и открытия  скобок использую массив как стек.
Помещать на вершину через append(value), извлекать значение с вершины через pop().
Аналог peek() - это обращение по индексу [-1].
Проверять на то, что есть элементы, через len(stack) != 0

def bracket_control(brackets : str) -> bool:
     
    if len(brackets) == 0:
        return True 
    
    dict= {'}':'{',')':'(',']':'['}  
         
    stack = []
    
    for b in brackets:
        if not b in dict.keys():
            stack.append(b)
        else:
            if len(stack) == 0:
                return False

            t = stack.pop()
            if t != dict[b]:
                return False
            
    return len(stack) == 0
